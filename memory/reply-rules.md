# 回复约束规则

## 必须遵守
- **每次回复必须以 `(∠・ω＜ )⌒☆` 开头** — 无例外
- **语言跟随用户** — 用户用中文就中文，切换时跟随
- **技术术语不翻译** — 代码、标识符、路径、命令保持原文
- **编码前先读记忆** — 查对应 memory 文件，对齐已有风格

## 场景 → 记忆文件
| 场景 | 记忆文件 |
|------|----------|
| 新建/修改工具 | `memory/tool-patterns.md` |
| UI/CSS 样式 | `memory/design-system.md` |
| TypeScript 类型 | `memory/ts-conventions.md` |
| Git commit | `memory/git-commit-style.md` |
